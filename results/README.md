## Data
- `step3_results.csv` — per (model × task) row for the 3-model run.
- `step3/results_<model>.json`, `step5/results_<arm>.json` — raw scored rows.
- `step5_probe_raw_<model>.json` — per-item IN/OUT P(IN) (the logit probe).
- `step5_calibrate.json`, `step5_crossmodel.json` — calibration sweep + cross-model table.
- `failure_mode_metrics.csv` — MUHAI evaluator behaviour on its own failure modes.
- `step3_taskset.json`, `step3_taskset_tagged.json` — the 50 tasks + coupled/benign tags.
- `step3/raw/`, `step5/raw/`, `pilot/`, `menu_pilot/` — saved prompts & model responses.

`runpod_*.json` (pod info) are git-ignored — they're regenerated by `runpod_deploy.py`.
